ハンズオン · v2026.5

自分専用の
ローカル翻訳 LINE Bot を
1 時間で作る

自分の PC で動くローカル LLM と AI エージェントを組み合わせて、スマホの LINE から日本語↔英語を翻訳できる Bot を作るハンズオン。Claude Code が伴走してくれるので、設定は対話するだけ。

⚡ 約 60 分 💰 Claude Code 以外ゼロ円 🔒 LLM はあなたの PC で動く 🖥 Mac / Win(WSL2) / Linux
🧠
Step 1

ローカル AI を作る

Ollama (or 同等) でローカル LLM を起動し、Hermes Agent で翻訳エージェントを CLI 上で動かすところまで。
構成は自由度高め。Claude Code と相談しながら自分の PC に合わせて調整します。

⏱ 約 20–30 分 🎯 CLI 翻訳が動けば完了
始める
💬
Step 2

LINE と繋げる

LINE Developers Console での手動操作と、Claude Code による自動セットアップを組み合わせて、スマホ LINE から翻訳できる状態に。
Step 1 が完了してから取り組んでください。

⏱ 約 20–30 分 🎯 スマホで翻訳が返れば完了
始める
🎨 完成後 — カスタマイズして遊ぶ

SOUL.md を書き換えるだけで、翻訳 Bot を「議事録要約 Bot」「英文添削 Bot」「キャラクター雑談 Bot」などに変身させられる。Slack/Discord 連携・LLM 差し替え・別フレームワーク (NanoClaw / OpenClaw) など、さらに進んだ挑戦のヒントも。

📖
ちなみに解説 — なぜ Hermes Agent を選んだのか?
OpenClaw / NanoClaw / Hermes Agent の違いを比較。今回 Hermes を選んだ理由 (シンプル & LINE 対応) を解説
→

🗺 全体アーキテクチャ

作るものの設計図。Step 1 で下半分、Step 2 で上半分を組み立てます。

[ スマホの LINE アプリ ] ↕ HTTPS (Webhook) [ LINE Messaging API ] ↕ [ Cloudflare Tunnel ] ← Step 2 で導入 ↕ [ Hermes Agent (Gateway) ] ← Step 1 で導入 └─ plugins/platforms/line (公式 LINE アダプタ同梱) ↕ OpenAI 互換 API [ Ollama (LLM ランタイム) ] ← Step 1 で導入 └─ qwen2.5:3b (推奨) or 他モデル ↕ これら全部を組み立てるのが ↑ [ Claude Code ] ← あなたの伴走パートナー
